# T-9.3 — `examples/demo.py` end-to-end

## Goal
`examples/demo.py` from T-9.1 must be runnable. This task adds an **automated** test that spawns it via `subprocess` and verifies exit code + key output substring presence.

## File to Create

### `tests/test_demo.py` (~80 lines)
```python
"""End-to-end test for examples/demo.py.

Spawns the demo as a subprocess, asserts exit code 0 and key output
substring presence.
"""
import subprocess
import sys
import pytest

EXAMPLE = "examples/demo.py"

def test_demo_exits_cleanly():
    proc = subprocess.run(
        [sys.executable, EXAMPLE],
        capture_output=True, text=True, timeout=30,
        cwd="/mnt/c/sml/project/py_project/tinydb",
    )
    assert proc.returncode == 0, f"stderr: {proc.stderr}"
    # Output includes some keywords from the demo script.
    out = proc.stdout
    assert "tinydb v0.1 demo" in out or "demo" in out.lower()
```

## Mechanical Verification

- `python -m pytest tests/test_demo.py -q` passes.
- `python /mnt/c/sml/project/py_project/tinydb/examples/demo.py` exits 0.
