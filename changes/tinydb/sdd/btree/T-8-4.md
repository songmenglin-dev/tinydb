# T-8.4 — Meta-commands

## Goal
In the REPL: `.exit`/`.quit` → exit 0; `.help` → print help; `.tables` → list table names; `.schema` → dump each table's CREATE TABLE.

## File
Modify `src/tinydb/cli/repl.py` to dispatch `.<cmd>` before trying SQL parse.

## Tests Required
`tests/cli/test_meta.py` (≥6 cases):

1. `.exit` → returns 0, prints goodbye.
2. `.quit` → returns 0 (alias).
3. `.help` → prints help text.
4. `.tables` → prints each table name in `db.catalog.list_tables()`.
5. `.schema` → prints full `CREATE TABLE ...` statement per table.
6. Unknown `.foo` → "unknown command" message; doesn't exit.
