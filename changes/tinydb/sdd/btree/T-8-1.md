# T-8.1 — Argparse for CLI

## Goal
Build the CLI argv parser. `python -m tinydb --db <path> -c '<sql>'` runs one command and exits. `python -m tinydb --db <path>` enters REPL.

## Files to Create

### `src/tinydb/cli/__init__.py` (≤ 30 lines)
Public exports: `parse_args`, `dispatch_one`, `dispatch_repl`, `format_rows`.

### `src/tinydb/cli/__main__.py` (≤ 50 lines)
The entry point: `if __name__ == "__main__": sys.exit(main())` where `main()` parses argv and dispatches.

### `src/tinydb/cli/argparse_ext.py` (≤ 100 lines)
The argparse parser. Three modes:
- `python -m tinydb --db <path> -c '<sql>'` → run one command, exit.
- `python -m tinydb --db <path> [--repl]` → enter REPL.
- `python -m tinydb --help` → print help and exit 0.
- `python -m tinydb --version` → print version and exit 0 (T-9 polish).

```python
def parse_args(argv: list[str] | None = None) -> Namespace:
    p = argparse.ArgumentParser(prog="tinydb", ...)
    p.add_argument("--db", required=True, help="Path to database file")
    p.add_argument("-c", "--command", help="Run one SQL command and exit")
    p.add_argument("--version", action="store_true")
    return p.parse_args(argv)
```
