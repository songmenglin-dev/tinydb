# T-9.2 — Coverage audit (gate ≥ 80% on whole `src/tinydb/`)

## Goal
Per-module coverage already holds (B1-B8 gates all passed). T-9.2 is the whole-codebase gate that B7.2 reported as DEVIATION: 74.80% < 80%.

Bring `src/tinydb/` to ≥ 80% coverage by adding targeted unit tests for the 3 modules identified at T-7.2 close:
- `src/tinydb/tx/wal.py` — 74% (CRC decode error paths untouched)
- `src/tinydb/types/codec.py` — 64% (variant encoding paths)
- `src/tinydb/types/coerce.py` — 42% (per-TypeTag coercion rules)

## Files to Create / Modify

### `tests/tx/test_wal_more.py` (NEW) ~120 lines
Tests targeting uncovered branches in `wal.py`:
- `WALCorruptionError` raised when a record payload's `[type|payload_len]` is corrupted.
- CRC truncation → detection.
- Truncated last-frame (e.g. write 5 frames then truncate to half) → `iter_from` stops cleanly.
- `truncate_to(N)` removes records ≥ N.
- Tail truncation after WAL restart → LSN recovery correct.
- New session over an existing WAL recovers LSN by walking frames.
- Frame header parsing error (e.g. unknown type byte).
- Mode 'w+b' open + close + reopen lifecycle.

### `tests/types/test_codec_more.py` (NEW) ~150 lines
Tests targeting `encode_value` / `decode_value` for ALL `TypeTag` variants:
- Encode/decode INT 8/16/32/64 if those exist; otherwise INT 64 (the one we have).
- Encode/decode Float (Float32 / Float64 if both).
- Encode/decode Bool True/False.
- Encode/decode Text (empty, single char, multi-byte UTF-8).
- Encode/decode Null / None.
- Encode/decode Json.
- Encode/decode Date / Time / DateTime if those exist.
- Encode/decode Decimal if exists.
- Encode/decode Blob (empty, 1 byte, multi-KB).
- Roundtrip: encode→decode for each type → result == original.
- Error: encoding a Python value of the wrong type (e.g., bool under Int) raises `_CodecError`.
- `value_size(value, tag)` for each type returns the right byte count.

### `tests/types/test_coerce_more.py` (NEW) ~120 lines
Tests targeting `coerce_value` for each (value, tag) combination:
- `int` (0, negative, large) → coerce to INT.
- `float` (0.0, negative, scientific) → FLOAT.
- `str` (empty, normal) → TEXT.
- `bool` True/False → BOOL.
- `None` → NULL.
- `dict`, `list`, `str-as-JSON` → JSON.
- Date/time/datetime strings → DATE/TIME/DATETIME.
- Decimal strings → DECIMAL.
- bytes → BLOB.
- Coerce failure: value that can't be coerced (e.g., "abc" for INT) raises `TypeMismatchError` or similar.
- Roundtrip: encode_row uses coerced values correctly.
