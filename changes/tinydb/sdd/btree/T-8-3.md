# T-8.3 — REPL core

## Goal
Read-eval-print loop. Read SQL from stdin line-by-line; for each non-empty line, execute and print result. Quit on EOF or empty input + explicit command (T-8.4 meta-commands).

## File
`src/tinydb/cli/repl.py` (≤ 150 lines)

```python
def run_repl(db: tinydb.Database, *, input_fn=input, output=print) -> int:
    """Drive the REPL. Returns process exit code (0 normally)."""
    output("tinydb v0.1 REPL — enter SQL, or '.help'")
    while True:
        try:
            line = input_fn("tinydb> ")
        except EOFError:
            return 0
        line = line.strip()
        if not line:
            continue
        # meta-commands (T-8.4) and SQL dispatch
        try:
            rows = db.execute(line)
            if rows:
                output(format_rows(rows))
        except ParseError as e:
            output(f"ParseError: {e.msg}")
        except TinydbError as e:
            output(f"Error: {e}")
    return 0
```
